## 自动化测试

执行命令：

```bash
.venv/bin/pytest
```

结果：

```text
11 passed
```

覆盖场景：

- OpenRouter 模型配置加载。
- 环境变量读取。
- 健康检查不泄漏真实 key。
- webhook memo UID 提取。
- webhook 幂等键生成。
- webhook 重放不重复创建任务。
- 管理接口 token 鉴权。
- jobs 查询。
- 任务 claim、failed、retry 状态流转。
- worker 使用模拟 Memos Client 处理 memo。

## 未完成的真实验收

真实 Memos 联调尚未完成，原因是当前 Linux 环境没有 Docker：

```text
docker: command not found
docker compose: command not found
```

在 Docker 可用或提供远程 Memos 实例后，需要补跑：

```text
手动创建 3 条 Memos memo
-> webhook 到达 Sidecar
-> 产生 3 个 process_memo 任务
-> worker 拉取真实 memo 详情
-> 全部任务成功
-> 重放同一 webhook 不产生重复任务
```
