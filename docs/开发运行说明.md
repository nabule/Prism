# 开发运行说明

## 当前阶段

当前实现范围为 P0+P1：

- FastAPI Sidecar API。
- Memos webhook 接收。
- SQLite 任务表、工作区表和 memo 映射表。
- 后台 worker 轮询处理 `process_memo` 任务。
- Memos API Client 基础封装。
- OpenRouter 模型配置预留。

AI 整理、标签审核、附件解析、RAG、自动文档和备份恢复属于后续阶段。

## 环境准备

本项目在 WSL/Linux 下开发和测试。当前系统 Python 受 PEP 668 保护，应使用本地虚拟环境：

```bash
python3 -m venv .venv
source .venv/bin/activate
python -m pip install -U pip
python -m pip install -e ".[dev]"
```

## 启动 API

```bash
source .venv/bin/activate
uvicorn memosima.api.app:create_app --factory --host 0.0.0.0 --port 8080
```

健康检查：

```bash
curl http://localhost:8080/health
```

## 启动 Worker

处理一条待处理任务：

```bash
source .venv/bin/activate
memosima-worker --once
```

持续轮询：

```bash
source .venv/bin/activate
memosima-worker
```

## 管理接口

查询任务：

```bash
curl -H "Authorization: Bearer $SIDECAR_ADMIN_TOKEN" \
  "http://localhost:8080/admin/jobs"
```

重试失败任务：

```bash
curl -X POST -H "Authorization: Bearer $SIDECAR_ADMIN_TOKEN" \
  "http://localhost:8080/admin/jobs/1/retry"
```
