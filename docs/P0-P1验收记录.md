## 未完成的真实验收

真实 Memos 联调尚未完成。镜像拉取阻塞已通过切换 Xget 域名解决：

```text
Docker Hub: failed to resolve reference "docker.io/neosmemo/memos:stable" / i/o timeout
xget.xi-xu.me Docker Hub mirror: unexpected status 429 Too Many Requests
xget.xi-xu.me GHCR mirror: unexpected status 429 Too Many Requests
xget.your-domain.com Docker Hub mirror: docker compose pull memos 成功
xget.your-domain.com GHCR mirror: npx nx build sidecar 成功
```

已按 Xget 规则将 Docker Hub/GHCR 镜像改为 `xget.your-domain.com/cr/...` 加速地址，后续需要继续执行 Compose 启动与真实 webhook 联调。

在 Docker 可用或提供远程 Memos 实例后，需要补跑：

```text
手动创建 3 条 Memos memo
-> webhook 到达 Sidecar
-> 产生 3 个 process_memo 任务
-> worker 拉取真实 memo 详情
-> 全部任务成功
-> 重放同一 webhook 不产生重复任务
```
