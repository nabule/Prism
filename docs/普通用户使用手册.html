Prism (棱镜) 普通用户工作流指南

🌈 Prism 的设计理念：
您随手丢进 Memos 的碎片笔记、手绘草图和脑图附件，就像一束朴素无序的 白光。当它穿过 Prism 棱镜 时，被 AI 离线优雅折射，化为层级大纲、精准标签、ToDo 待办以及可供一键编译的超级 Prompt。零星想法进来，绚丽彩虹出去。

作为日常记录者，您不需要接触任何复杂的代码、Docker 容器或命令行。您只需要在您喜爱的 Memos 客户端中像平时一样自然地记录、粘贴即可。Prism 会在后台自动完成所有的精炼与整理。

🎯 核心功能一览表

核心功能 它的用途 解决您的痛点
🌈 AI 智能整理 为凌乱备忘提炼核心标题、整理摘要、生成执行 To-Do。 “写时一时爽，过后火葬场”。海量两句话的闪念卡片由于缺乏整理，几天后就变成了查找不到的数字垃圾。
🏷️ 标签双层治理 自动区分已有正式标签与临时新标签，人工一键审核才入库。 AI 自动分类胡乱发明标签（导致标签泛滥），或用户自己手写标签不统一（如 #部署 与 #deploy 并存）。
💻 图表离线高保真解析 100% 本地离线解密 Draw.io 流程图和思维导图大纲，转换为 Markdown 文本。 上传的脑图、流程图对传统搜索工具来说完全是“乱码和黑盒”，根本无法检索其内部的文本和逻辑。
🔍 离线 QA & Prompt 编译器 基于精确标签与正文双路匹配，一键拼装超级 Prompt。 抗拒将全部个人日记和商业密码暴露给云端 AI 厂商，但又极度需要专属知识库问答。
⏰ 时间智能提醒 抽取正文自然语言时间，到期自动向手机发送 Bark 表单消息。 备忘里随手写了待办却容易忘记，需要一个极简的系统推送闹钟。

1. 🌈 AI 智能整理功能

📌 功能用途

当您在 Memos 中保存了一条随笔或资料摘录后，Prism 会在后台自动调用大语言模型，对内容进行深度结构化分析，提炼出“标题”、“摘要”、“核心要点”以及**“待办事项”**，并自动生成一条新的 AI整理 笔记，通过 Memos 的引用链条与您的原笔记进行双向绑定。

💡 解决的痛点：解决了碎片化笔记“易记录、难整理、无法重用”的问题，将无序的想法自动变成了可以直接指导下一步行动的高清闪念卡片。
🔄 日常工作流嵌入：
  1. 正常记录：如平时一样在手机或电脑端 Memos 记下简短的想法，然后点击保存。
  2. 秒级生效：1至2秒内，该笔记的下方会自动生成一个“引用连接”，点开即可浏览 AI 帮您梳理的规范结构体。
  3. 执行待办：可以直接在 AI 整理的 Task Checklist 中勾选待办，追踪事项进度。

💡 优质记录范例：

今天验证了 Prism 的 Mind Elixir 附件解析。
把 JSON 文件递归遍历并秒级转换为了带缩进的 Markdown，比大模型在线 OCR 快得多。
下一步要写完 test_p3_qa_drawio 单元测试并跟进 master 分支的推送。

2. 🏷️ 标签双层治理与自动建议

📌 功能用途

Prism 将标签分为三类：正式标签 (Active)、候选标签 (Candidate)、禁用标签 (Disabled)。

  • 如果您在写笔记时手写了已有正式标签，Prism 会直接归档。
  • 如果您完全不写标签，AI 会根据语义为您推荐已有的正式标签；若 AI 认为需要创立新标签，该新标签将进入“候选标签”列表，必须由您在控制台点击 Approved，才会升级为正式标签。
💡 解决的痛点：杜绝了 AI 自动生成导致的标签污染，并提供了标签同名跨层级归一能力（如自动将同名的 #数管 自动合并至已有层级 #项目/数管），确保您的知识库标签树永远干净清爽。
🔄 日常工作流嵌入：
  1. 大胆记录：写笔记时不必纠结打什么标签。如果已有确定标签可以直接手写（例如 #项目/ Prism），如果没有，直接留空发布即可。
  2. 标签审核：每周或定期进入 Prism 控制台：`http://localhost:8080/admin/ui#tag-candidates`，查看 AI 从您这周备忘中提炼的“候选标签”，点击“通过”入库，或者“拒绝”丢弃。

3. 💻 流程图与思维脑图本地离线解析

📌 功能用途

如果您在笔记中上传了 `.drawio` (或带有 XML 数据的 `.drawio.svg`) 流程图文件，或者是 `Mind Elixir` 思维导图脑图的 JSON 导出文件，Prism 后台会自动在本地 100% 离线对其解压与还原，提取出其中所有的纯文本节点、节点坐标关系，并重塑为排版精致、带 Markdown 缩进大纲的大纲大纲。

💡 解决的痛点：打破了“非结构化附件不可读”的瓶颈。无需消耗任何昂贵的在线 OCR 费用或 LLM API Token，图表附件在本地安全变成纯文本，可供 AI 直接理解和全局搜索。
🔄 日常工作流嵌入：
  1. 直接上传：在 Memos 撰写笔记时，直接拖入或上传您的脑图 `.json` 或 Draw.io 流程图。
  2. 自动丰富：如果您只传了附件而未写正文，Prism 整理完后会自动在 Memos 原笔记中回写一个清晰的卡片标题（如 # Prism 架构设计），极大提升阅读辨识度。

4. 🔍 离线 RAG 与超级 Prompt 编译器

📌 功能用途

在 Prism 管理页面的 “QA 离线问答” 面板中，您可以通过拼音实时下拉筛选特定的业务标签，输入您想向大模型提问的内容。Prism 会联合检索出这批标签下的所有原始笔记、AI 整理摘要和已解析附件 Markdown，并将其作为 Context 离线组装为一条带有完整层级的超级 Prompt，供您一键复制。

💡 解决的痛点：完美的本地沙箱隔离方案。您无需把几十万字的个人笔记和敏感工作密码交出去托管，仅仅在本地自动拼好 Prompt，带入已经验证安全的网页端沙箱大模型（如官网 DeepSeek / ChatGPT）中使用，把数据主权牢牢抓在自己手里。
🔄 日常工作流嵌入：
  1. 提出问题：打开管理端页面 http://localhost:8080/admin/ui#qa。
  2. 组合标签：在胶囊选择器中打字，快速回车添加几个知识主题标签（例如 部署、Caddy）。
  3. 一键编译复制：输入您的查询问题（例如：“如何在家庭服务器上一键迁移？”），点击编译，随后点击 **“一键复制”**。
  4. 粘贴获知：在新标签页打开外部大模型，Ctrl+V 直接粘贴即可获得深度融合您私人知识库的精准回答。

5. ⏰ 时间智能提醒系统

📌 功能用途

只要您在 Memos 的正文里写下了 #提醒 标签加一段普通的时间描述，Prism 后台会自动为您建立定时闹钟，到点后会向您的配置终端（如手机 Bark 推送 App）精确推送一条带标题和正文的表单通知。

💡 解决的痛点：不需要去专用的 To-Do 软件中繁琐地建卡片和调时间。随手写备忘的同时，闹钟和推送自动生成，绝不遗忘。
🔄 日常工作流嵌入：
  1. 极速建闹钟：在 Memos 随手写下一行字： 
    #提醒 明天 14:30 跟进客户合同签订
  2. 自动诊断：如果时间模糊（例如只写了“有空提醒我”）或时间已过期，Prism 会在您原笔记下方自动发一条评论，温和提醒您需要补齐具体时间。
  3. 接收推送：明天下午 14:30 分，您的手机会准时收到 Bark 推送消息，点击即可在手机上直达原始 Memos 页面。

6. 👥 团队协作（家庭 / 小组共享知识）

📌 功能用途

如果您和家庭成员、研发小组、运营搭子需要共享一小块知识（菜谱、事故复盘、话术模板），但又不想把整个个人库公开 —— Prism 提供一层「团队知识库」：管理员开团 → 生成邀请码 → 成员到 /teams/ui 自助加入，全程无需任何命令行。

💡 解决的痛点：以前要么把整个 Memos 共享出去（牺牲隐私），要么再独立部署一套（运维翻倍）；现在用同一个 Sidecar 即可按团队物理隔离词条、向量、邀请、角色，一份运维多份用。

详细的开团/邀请/角色/最后 owner 保护等高级用法请参考 团队知识库使用手册。

7. 🔒 隐私与 SaaS 多租户隔离提示

警示：本系统在单实例下绑定单一 API 密钥，且全量知识解析存于同一个 SQLite 数据库中。请勿与他人共享使用同一个 Sidecar 实例，否则会发生跨用户的知识数据检索与泄露。

最佳 SaaS 部署方案：如果家庭成员或团队中有多人记录需求，管理员应在宿主机上使用独立文件夹为每位成员运行一套物理隔离的容器栈，并在各自的 .env 中指定不同的 GATEWAY_PORT 端口启动，以物理级别的沙箱隔离方式保障大家的数据主权。

8. 💾 一键搬家与备份

当您更换了新服务器、升级系统或者需要物理归档时，管理员可以在 Prism 控制台的 “备份恢复” 标签页中一键下载整站的 Sidecar 快照 ZIP 备份。

配合生产端提供的 [deploy.sh](file:///home/abc/code/memosima/deploy.sh) 脚本，您可以在新的机器上解压后一键热启动，秒级完成无缝异地完美搬家！
