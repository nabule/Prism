Memosima

Memosima 是基于 Memos 的个人 AI 知识库 Sidecar 服务。Memos 负责采集和保存原始知识，Sidecar 负责 AI 整理、标签治理、澄清评论、附件解析和后续检索能力。

当前状态

已完成 P0、P1、P2 核心闭环，并开始 P3 文本附件解析基础。

  • Memos webhook 接入和 SQLite 任务系统。
  • AI 整理 memo 创建，支持 OpenRouter/OpenAI-compatible LLM 和本地模板回退。
  • Memos 原生 REFERENCE relation 关联原始 memo 与 AI memo。
  • 内置 /admin/ui 调试管理页面，用于查看任务、重试任务和审核候选标签。
  • 候选标签审核和审核后 active 生效。
  • LLM 可从正文自动建议正式标签和候选标签；新标签仍需人工审核。
  • LLM 提示词从 config/prompts.yaml 加载，管理页面可保存默认提示词，也可在重试时临时覆盖。
  • 待澄清评论和 waiting_user 状态。
  • .txt、.md 附件解析为 artifact。

未完成范围：docx/xlsx/pdf/drawio/Mind Elixir 专用解析、向量索引、问答、自动文档、备份恢复。

快速开始

cp .env.example .env
npx nx run sidecar:up
curl http://localhost:8080/health

调试管理页面：http://localhost:8080/admin/ui。页面不会读取服务器端密钥，需要手动输入 SIDECAR_ADMIN_TOKEN，并只保存在当前浏览器的 localStorage。页面也可以编辑默认 LLM 提示词，或在重试任务时只临时覆盖当前任务使用的提示词。

常用命令

npx nx test sidecar
npx nx run sidecar:compile
npx nx build sidecar
npx nx run sidecar:probe-memos
